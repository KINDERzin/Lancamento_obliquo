## FORMULAS E VALIDAÇÕES

EIXO X
  V = v0 + at
  V²= v0² + 2aΔt -> Torricelli
  X = x0 + v0t + (at² / 2)
  Δx = x - X0
  Δx  = posição percorrida
  x   = posição final
  x0  = posição inicial
  vx  = velocidade final (V ou Vx)
  V0x = velocidade inicial
  a   = aceleração
  t   = tempo (t)

  Velocidade final (Vfinal)
    - Vfinal = Vinicial + (aceleração * tempo)

  Distância percorrida (ΔX)
    - ΔX = Xfianl - Xinicial


EIXO Y
  Vy = v0y - gt
  Y  = y0 + v0yt - gt²/2
  Vy²= v0y² - 2gΔy 
  Δy = y - y0

  Δy  = altura percorrida
  Y   = altura final
  y0  = altura inicial
  Vy  = velocidade final
  V0y = velocidade inicial
  g   = gravidade
  t   = tempo
